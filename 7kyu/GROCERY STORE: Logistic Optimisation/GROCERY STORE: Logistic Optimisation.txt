You have to find out how many unique products have each of the Producer. If you take only few items from some of them you will 
stop going there to save the gasoline:)

In the results show producer and unique_products which you buy from him.

Order the result by unique_products (DESC) then by producer (ASC) in case there are duplicate amounts,

products table schema
* id
* name
* price
* stock
* producer

results table schema
* unique_products
* producer
